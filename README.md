## Setup Environment - Shell/Terminal
```
mkdir proyek_analisis_data_dicoding
cd proyek_analisis_data_dicoding
pipenv install
pipenv shell
pip install -r requirements.txt
```

## Run steamlit app
```
streamlit run dashboard.py
```
